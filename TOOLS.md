# AVAILABLE TOOLSET
- **`gemini`**: CLI tool for cloud queries. Syntax: `gemini "prompt"`
- **`python3`**: Local execution.
- **`mcp`**: Model Context Protocol for local file operations.
- **`ralph_loop.sh`**: The main autonomous loop entry point. Usage: `./ralph_loop.sh [--sandbox] "Objective"`
- **`ralph_core`**: The Python package containing the agent swarm logic.
